# LiveBov

A stock tracking app capable of displaying info about stocks traded on [BOVESPA(Bolsa de Valores de SÃ£o Paulo)](https://www.b3.com.br/en_us/). Uses the [brapi](https://brapi.dev) API in order to get current info.
